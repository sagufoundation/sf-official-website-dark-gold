# SAGU Foundation Official Website (Dark Gold)
Website profil resmi SAGU Foundation dengan tema dark-gold, fokus pada informasi program, mitra, galeri kegiatan, dan formulir kontak.

## Demo
https://sagufoundation.github.io/sf-official-website-dark-gold/

## Fitur Umum
- Topbar dan navbar modern dengan struktur menu organisasi.
- Dropdown navigasi untuk kategori program utama.
- Hero section interaktif dengan slider konten.
- Section **Available Courses** dengan card poster kursus.
- Halaman khusus `courses.html` dengan:
  - daftar kursus terbaru,
  - sidebar pencarian,
  - filter tahun,
  - filter term 1–4,
  - daftar kategori kursus.
- Section partner dengan tampilan kategori.
- Gallery kegiatan dengan tampilan lightbox.
- Form kontak lengkap (nama, nomor telepon, email, topik, pesan, konfirmasi, kirim).
- Desain responsif untuk desktop dan mobile.

## Struktur Halaman
- `index.html` — halaman utama.
- `courses.html` — halaman daftar kursus.
- `images/` — aset gambar website.
